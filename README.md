## ✨ Features

- 🎯 **15 unique games** across multiple genres
- 📱 **Mobile-friendly** — touch controls for all games
- 🔊 **Procedural audio** — Web Audio API synthesized sound effects
- 💾 **Save system** — progress, stats, and achievements persisted locally
- 🏆 **Achievements** — unlock challenges across games
- ⚡ **Zero dependencies** — pure vanilla HTML5/Canvas/JavaScript
- 🎨 **Pixel-art style** — handcrafted procedural graphics
- 🌐 **Offline ready** — no server required, runs entirely in-browser

## 🎯 How to Play

1. Visit **[Hải Game World](https://leminhhai13603.github.io/HaiGameWorld/index.html)**
2. Pick a game from the grid
3. Play and have fun!

**Keyboard shortcuts** (most games):
- `WASD` / `Arrow Keys` — Movement
- `Space` / `Click` — Action
- `P` / `Esc` — Pause
- `R` — Restart

---

## ⚡ Performance

All games are optimized for smooth 60fps gameplay:

- **Write-index filtering** — O(n) array cleanup instead of O(n²) splice
- **Particle limits** — bounded particle systems prevent frame drops
- **AudioContext cleanup** — proper resource management across sessions
- **Canvas optimization** — minimal state changes, efficient rendering
